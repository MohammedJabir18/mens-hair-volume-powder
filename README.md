## 🛠️ The Tech Spec Sheet

*   **Substrate (HTML5):** 100% semantic structures utilizing `<data>`, `<samp>`, and `<output>` variables. Strict ARIA accessibility landmarks (`role="radiogroup"`, `role="radio"`) ensure keyboard navigational compatibility.
*   **Design Tokens (Vanilla CSS3):** obsidian black `#000000`, card backing `#0D0D0D`, raw unbleached borders, and aviation crimson `#E11D48`. Programmed using CSS custom properties with zero bloated utility frameworks.
*   **Active Logic (Vanilla ES6):** Draggable before/after clipping system powered by high-performance `clip-path` inset coordinates, cached bounding rect coordinates to eliminate layout thrashing, and active stock count periodic decrimentation triggers.
*   **Motion Engineering (GSAP 3.12):** Timeline-staggered entry animations utilizing ScrollTrigger limits. Fully accessible; `prefers-reduced-motion` media queries override movement, gracefully scaling back to fast linear opacity fades.
